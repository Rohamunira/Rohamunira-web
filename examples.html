Information and Destination

Teresmokenburg project are web crawl project that target URL for destination search new website but you don't had a keywords to search the website, this project will search URL or Link from site to site (this help somebody that need new website without keywords to search).

Process

First you should had python installed in your computer or operating system, then run it with example

>> python main.py --urls "https://playwright.dev" --chromium "chrome-win/chrome.exe" --output "output.txt" --headless "False"

The --urls is for first urls that will targetted, the --chromium is the path to browser file, for these version only receive chromium browser, the --output is the output path, the --headless is specify are browser should displayed or not

After you run these code in your cmd or terminal then this will open the browser with opening page from parameter --urls first, then the browser will search list urls from first parameter value page, after it it check the first urls that found in the list urls if the urls was in the file that specified in --output, it will not written in the file. else it written in the file and opening the first urls that in list. the it will doing again from first time but new with new urls (looping)
